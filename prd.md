# 低空飞行管理平台需求规格说明书 (PRD)

> **版本**：v2.0  
> **状态**：正式发布  
> **密级**：内部公开  
> **最后更新**：2026-01-26

## 1. 项目愿景与目标

### 1.1 产品定位
本平台致力于成为**低空产业生态的核心中枢**。通过深度融合大数据、人工智能、三维GIS与物联网技术，构建“云-网-端”一体化的低空飞行服务系统，实现对低空资源的精细化管理与高效利用。

### 1.2 核心价值
- **全生命周期闭环**：覆盖从“需求-规划-审批”到“执行-监控-分析”的全流程。
- **数智化决策**：基于多源数据融合与智能算法，提供自动航线规划、动态空域适配与风险预警。
- **人机高效协同**：通过标准化流程与智能调度，最大化“人、机、环境”的协同效能。

## 3. 功能架构 (Core Modules)

系统由七大核心业务模块构成，形成业务闭环。

### 3.1 空域航线管理 (Airspace & Route Management)
*打通空域资源的“申报-规划-适配”全链路，保障飞行合规与高效。*

- **空域数据中心**：
  - 整合低空试点空域、航线及临时起降点数据，构建全域三维空域栅格图。
  - 实时可视化展示空域状态（开放/占用/管制/禁飞）。
- **智能航线规划**：
  - **基于地图基座**：结合地形（DEM）、建筑模型（3D Tiles）与气象数据。
  - **算法引擎**：自动生成避障、能耗最优的飞行路径；支持禁飞区/限制区自动校验。
  - **动态重规划**：遇临时交通管制或突发天气时，自动推荐备选航线。
- **空域审批电子化**：
  - 在线提交飞行计划（时段/空域/机型），对接军民航管系统自动获取批复凭证。

### 3.2 飞行任务管理 (Flight Task Management)
*实现任务从“创建”到“归档”的精细化流转。*

- **任务智能分配**：
  - 基于任务类型（巡检/物流/救援）、优先级、地理位置。
  - 算法自动匹配最优“无人机 + 飞手 + 电池”资源组合。
- **全流程监控**：
  - 实时追踪任务进度（待命-起飞-作业-返航-完成）。
  - 支持任务中途挂起、取消或变更。
- **成果自动归档**：
  - 任务结束后，自动关联航拍影像、遥测数据与分析报告，形成任务档案。

### 3.3 无人机飞控中心 (Flight Control Center)
*远程操控与实时监管的“大脑”。*

- **沉浸式驾驶舱**：
  - **多视角切换**：支持第一人称（FPV）、第三人称及地图上帝视角的无缝切换。
  - **实时遥测**：毫秒级回传高度、速度、姿态、链路信号强度。
- **智能辅助飞行**：
  - **环境感知**：复杂地形（山区/楼宇）下的自动避障与地形跟随。
  - **应急接管**：低电量/失联自动返航、悬停；支持指挥员远程切入拥有最高控制权。
- **数字孪生仿真**（保留 Demo 特性）：
  - 在真实飞行前，基于物理引擎进行全流程模拟，验证航线安全性与电量消耗。

### 3.4 低空设备管理 (Device Lifecycle Management)
*保障物理资产的高可用性。*

- **电子台账**：
  - 记录设备全生命周期信息（采购、参数、固件版本、维修记录）。
- **健康监测**：
  - 实时监控电池循环次数、电机温度、传感器健康度。
  - **预测性维护**：基于使用时长自动生成保养工单，杜绝“带病作业”。
- **智能调度**：
  - 根据设备剩余作业能力（续航/载重）与任务需求进行最优撮合。

### 3.5 低空智联网管理 (Low-Altitude IoV)
*构建“空-地-网”一体化感知网络。*

- **多源感知融合**：
  - 接入地面雷达、ADS-B、光电吊舱、气象微站等多源数据。
  - 智能识别非合作目标（“黑飞”无人机），提供轨迹预测与告警。
- **态势一张图**：
  - 全局展示辖区内所有飞行器的密度、流向与热力分布。

### 3.6 工单管理 (Work Order System)
*连接需求端与执行端的服务桥梁。*

- **敏捷派单**：
  - 支持客户发起临时/紧急需求（如故障报修、应急搜救）。
  - 按“就近原则 + 能力匹配”自动派单。
- **服务SLA监控**：
  - 监控响应时长、处理时效；超时自动升级预警。
- **质量画像**：
  - 基于用户评价与作业数据，建立服务质量评分体系，反哺调度策略。

### 3.7 人机协同管理 (Human-Machine Collaboration)
*优化生产关系，提升作业效能。*

- **资质画像**：维护飞手执照等级、擅长领域（夜航/巡检）及历史绩效。
- **SOP标准化导航**：
  - 作业过程中，系统自动推送标准操作程序（SOP）检查单，确保规范作业。
- **效能评估**：通过数据分析人机配合效率，识别短板并提供培训建议。

## 4. 技术架构规划

### 4.1 总体架构
采用 **微服务 + 云原生** 架构，前端采用 Vue3 + TypeScript + Mars3D。

```
[展示层 (Web/大屏/移动端)]
       ⬇
[网关层 (API Gateway / Auth)]
       ⬇
[业务中台] -> [空域服务] [任务中心] [设备中心] [工单中心]
[数据中台] -> [时空数据库] [轨迹分析] [多源融合]
[IoT 基座] -> [MQTT/WebSocket] [协议解析] [边缘计算]
       ⬇
[基础设施 (K8s / GIS Server / DB)]
```

### 4.2 关键技术指标
- **实时性**：遥测数据延迟 < 200ms，控制指令延迟 < 100ms。
- **并发性**：支持单区域 10,000+ 架次无人机并发在线接入。
- **渲染性能**：三维场景加载 < 3秒，大场景帧率稳定 > 30FPS。
- **可靠性**：核心服务可用性 99.99%，数据零丢失。
